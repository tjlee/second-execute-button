<!-- Plugin description -->
Adds a second execute button to the DataGrip / IntelliJ Database Console toolbar that runs all statements in the script as **separate statements** (one by one), regardless of the current execution mode setting.

Useful when your default execution mode is set to run the whole script as a single statement, but you occasionally need to execute each statement individually without changing the global setting.
<!-- Plugin description end -->
